# 🧠 Medical Research Assistant

## 📌 Overview
This project is a full-stack Medical Research Assistant that aggregates data from multiple scientific sources and presents structured insights to users.

## 🚀 Features
- Multi-source data integration (PubMed, OpenAlex, ClinicalTrials)
- Intelligent ranking of research papers
- AI-style summarized insights
- Clean and responsive React UI
- REST API backend using Node.js & Express

## 🛠 Tech Stack
Frontend:
- React
- Axios
- CSS

Backend:
- Node.js
- Express

APIs:
- PubMed API
- OpenAlex API
- ClinicalTrials API

## ⚙️ How to Run

### Backend
```bash
cd backend
npm install
node server.js
